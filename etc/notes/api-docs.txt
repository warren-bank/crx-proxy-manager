API:
====
  FindProxyForURL = (url, host) => configuration
where:
  url  = URL being accessed
  host = hostname extracted from the URL (for convenience)
  configuration = null || string containing building blocks, each separated by a semicolon

notes:
======
  * FindProxyForURL(url) doesn't provide any context
    - tabId   => where request was made
    - referer => url of window.document where request was made

--------------------------------------------------------------------------------

conclusions:
============
  * the Firefox API will allow a background script to easily identify the tabId where each request was made
    - this is required to support enabling the proxy on individual tabs

  * the Chrome API does not provide this granular visibility
    - Chrome will not be supported!
